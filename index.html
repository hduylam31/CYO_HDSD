Guideline Cài Đặt

📋
Thứ tự bắt buộc: Node.js → MCP Config → Skill
Phải cài Node.js hoàn tất trước mới chạy được MCP server. Sau khi MCP hoạt động mới setup Skill trong Claude.
🟢
Bước 1 — Cài Node.js
Runtime môi trường · Làm trước tiên
  • 1
    Tải file .msi (Windows) hoặc .pkg (Mac)
    → nodejs.org/en/download · Chọn LTS
  • 2
    Mở file vừa tải, nhấn Next liên tục đến khi xong
    Giữ mọi tùy chọn mặc định
  • 3
    Báo IT biết đã cài xong
    IT sẽ setup MCP server và kết nối Claude Desktop cho bạn
⚙️
Bước 2 — Kết nối MCP vào Claude Desktop
Edit config JSON · Sau khi Node.js xong
  • 1
    Mở Claude Desktop → nhấn tên mình → Settings → Developer → Edit Config
    Dùng Claude Desktop app (claude.ai/download), không phải browser
  • 2
    Paste JSON config bên dưới (thay thế toàn bộ nội dung cũ)
    {
      "mcpServers": {
        "cyo-cdp": {
          "command": "node",
          "args": ["/path/to/cyo-mcp/index.js"],
          "env": { "BQ_PROJECT_ID": "yody-prod" }
        }
      }
    }
    ⚠️ Thay /path/to/ bằng đường dẫn thực tế trên máy
  • 3
    Lưu file → Quit hoàn toàn Claude Desktop (Cmd+Q / Alt+F4) rồi mở lại
    Gõ thử: "Chạy health check MCP server" — Claude gọi được tool là thành công ✓
✨

  • 1
